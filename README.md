# tetris-clone
A tetris clone developed in JavaFX. Use the keyboard left arrow button to move left, respectively right arrow button to move shape right. To rotate the shape, press the up arrow button.

## Usage
To play the game, run the tetris.jar file, or else run the Main class to play.
